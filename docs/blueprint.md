# **App Name**: Stumble Studio

## Core Features:

- Camera Integration: Integrate camera functionality to capture photos directly within the web app.
- Real-time Filters: Apply filters to the camera feed in real-time, using map filter concepts.
- Frame Overlay: Overlay different frames on the camera view.
- Sticker and Text Addition: Add stickers and custom text to photos.
- Photo Gallery: Display taken photos in a gallery with the ability to download.
- AI background Removal Tool: Use AI to identify people in an image, then erase the background to make it transparent; allow the user to download the resulting image. This requires a tool with reasoning to correctly determine whether to activate based on image features.

## Style Guidelines:

- Primary color: Stumble Blue (#00a2ff) to evoke a playful and dynamic feel.
- Background color: Dark radial gradient from #222d5a to #0d1127 to provide contrast and depth.
- Accent color: Stumble Yellow (#ffcc00) for interactive elements and highlights to draw attention.
- Headline font: 'Fredoka One', cursive, with text shadows, for headlines.
- Body font: 'Nunito', sans-serif, for body text, buttons, and functional elements.
- Use playful and bold icons, possibly sourced from Stumble Guys game assets, where applicable, in bright colors to match the brand.
- Use chunky cards with thick borders to contain elements and give a bold appearance, and arrange them in a responsive grid layout.
